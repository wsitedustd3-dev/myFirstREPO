# ✈️ TripPick — 원클릭 여행 플랜 생성기

> "고민은 TripPick이, 짐은 당신이!"  
> 목적지·동행·테마·날짜·예산을 선택하면 맞춤형 여행 일정을 즉시 생성하는 웹 서비스

---

## 🗂️ 프로젝트 구조

```
index.html                  — 메인 페이지 (전체 UI 포함)
css/
  style.css                 — 전체 스타일 (다크/라이트 모드 포함)
js/
  main.js                   — 플랜 생성 로직, 달력, 지도, 유효성 검사
  travelData.js             — 국내 도시 여행 데이터
  overseas-data.js          — 해외 도시 여행 데이터 (50+ 도시)
  overseas-data-patch.js    — 데이터 패치 및 누락 도시 추가
README.md
```

## ✅ 구현 완료 기능

### 🧭 플랜 생성
- **목적지 선택**: 국내 + 해외 14개국 60+ 도시 칩 선택
- **동행 선택**: 혼자, 연인, 친구, 가족, 동료, 부모님 + 직접 입력
- **테마 선택**: 맛집탐방, 포토스팟, 액티비티, 힐링, 역사탐방, 쇼핑 등 12가지
- **교통수단 선택**: 복수 선택 지원
- **날짜 선택**: 달력 UI(시작일 기준), 박수 자동 계산
- **예산 슬라이더**: 20만~500만원, 5단계 티어 자동 분류
- **폼 유효성 검사**: 미선택 항목 shake 애니메이션 + 토스트 알림

### 📅 날짜별 독립 동선 (핵심 기능 — v2026.04.02 업데이트)
- **첫째날 · 중간날**: 각 5~6개 스팟 (겹치지 않는 완전히 다른 동선)
- **마지막날**: 3개 스팟 + 귀가 스팟 (숙박 없음)
- **숙박 스팟**: 날마다 동일 숙소 이름 · 동일 1박 가격으로 고정
- **스팟 중복 방지**: `usedNames` Set으로 전체 날짜 공유, 같은 장소 절대 반복 없음
- **예산 배분**: 숙박비 먼저 확보 → 나머지를 활동비로 배분
- **days 배열 지원 도시**: 제주도, 부산, 강릉, 경주, 여수, 전주, 속초, 통영, **대전** (각 날짜별 6개 스팟 데이터 보유)

### 🎲 미스터리 트립
- 예산만 설정하면 랜덤 목적지 + 테마 여행 플랜 생성

### 📋 플랜 결과 표시
- 여행 일정 타임라인 카드 (시간·이모지·장소명·설명·분위기·비용)
- 총 예상 비용 표시
- 동행·테마·교통수단 배지
- **데이터 준비 중 알림**: 전용 데이터 없는 도시 선택 시 대표 도시 기반으로 플랜 생성 + 알림 배너 표시

### 🔄 Reroll 기능
- 개별 장소 교체 (🔄 버튼)
- 전체 일부 Reroll
- 등급별 교체 풀 데이터 제공

### 🗺️ 지도 탭
- 여행 플로우 / 실제 지도 탭 전환
- Leaflet + OpenStreetMap 기반 마커 표시

### 🎉 일정 확정
- 확정 모달 + 컨페티 애니메이션

### 🌙 다크/라이트 모드
- 토글 버튼, localStorage 저장, 플리커 없음

### 📱 반응형 디자인
- 모바일/태블릿/PC 대응

## 🐛 수정 이력 (v2.1)

### 수정된 버그
1. **overseas-data.js 손상 블록 제거** — 1783~1832줄의 '그라나다' 키에 싱가포르 데이터가 혼입된 블록 완전 삭제
2. **JS 파싱 오류 해결** — 손상 블록으로 인한 `Unexpected token` 오류 해결
3. **patch.js 타이밍 문제 수정** — `DOMContentLoaded` 이벤트 의존에서 즉시 실행(IIFE)으로 변경
4. **폴백 로직 개선** — 해외 도시 데이터 없을 때 국내 랜덤 폴백 → 같은 나라 대표 도시 플랜 + 알림 배너
